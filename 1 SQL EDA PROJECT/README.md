# 📊 SQL EDA on Data Warehouse Project

This project focuses on performing **Exploratory Data Analysis (EDA)** using SQL on a structured Data Warehouse.

The goal is to extract meaningful insights from raw data and transform it into business-relevant information using SQL queries.

---

## 📌 Project Overview

In this project, I worked on data that follows a **Medallion Architecture** approach:

- 🥉 **Bronze Layer** – Raw data ingested from CSV files  
- 🥈 **Silver Layer** – Cleaned and transformed data  
- 🥇 **Gold Layer** – Analytics-ready data (Fact & Dimension tables)  

The analysis is primarily performed on the **Gold layer**, which is optimized for reporting and insights.

---

## 🧭 EDA Workflow

![EDA Workflow](./docs/eda_workflow.png)

The analysis in this project follows a structured EDA approach:

1. **Database Exploration**  
   - Understanding tables, schemas, and relationships  

2. **Dimensions Exploration**  
   - Analyzing categorical data (customers, products, etc.)  

3. **Date Exploration**  
   - Understanding time-based trends  

4. **Measures Exploration**  
   - Working with numerical metrics (sales, revenue, etc.)  

5. **Magnitude Analysis**  
   - Comparing values across categories  

6. **Ranking Analysis**  
   - Finding Top N and Bottom N entities
