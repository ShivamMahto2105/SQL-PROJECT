#### 📌 Description  
This project focuses on performing **Exploratory Data Analysis (EDA)** using SQL on a structured Data Warehouse.

The dataset is organized using a **Medallion Architecture (Bronze, Silver, Gold layers)**, where:
- Raw data is ingested from CSV files  
- Cleaned and transformed into structured tables  
- Modeled into analytics-ready format  

Using SQL, I explored the data to:
- Understand data distribution  
- Analyze business metrics and KPIs  
- Identify trends and patterns  
- Generate insights for decision-making
